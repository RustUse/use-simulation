# use-simulation-result

Summary helpers for finite simulation result series.

## Install

```toml
[dependencies]
use-simulation-result = "0.0.1"
```

## Scope

Use `use-simulation-result` when you need simple summary statistics and bounds
over finite numeric result series from simulation runs.

## RustUse workspace

This crate is part of the `use-simulation` workspace. Depend on
`use-simulation` when you want the full facade.

## Status

This crate is experimental and may evolve while the workspace remains below
`0.3.0`.
